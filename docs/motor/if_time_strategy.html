基于时间的开环强托策略

IF启动中的theta角度​

为了满足park与anti-park的角度需求，我们这里是人为造了一个角度。在实际的IF启动中，我们会对常数进行积分，得到omega，然后在积分一次，得到theta角度。做两次积分主要是保障升速用的。对于一些低转速的电机，这个常数c不要超过100

转子预定位及启动​

  1. idi_{d}id​=0，设定iqi_{q}iq​=iq∗i_{q}^{*}iq∗​
  2. 另iqi_{q}iq​=iq∗i_{q}^{*}iq∗​的同时，保持一段时间
  3. 设定常数c,让常数c积分得到omega，在积分得到θ\thetaθ

基于时间的切换补偿​

升速的过程由常量C控制，电机在IF工况下速度达到15%~20%，此时切换带来的负面影响最小。在IF运行至观测器收敛后，一般2s到3s就足够了。同时，在切换前0.1s内的任意时刻，记录dq轴的q轴电流值：

qreal=iq×cos(θe^−θe∗)q_{real} = iq \times cos(\hat{\theta _{e}} -\theta _{e}^{*} )qreal​=iq×cos(θe​^​−θe∗​)

其中θe^\hat{\theta _{e}}θe​^​是观测器的值，θe∗\theta _{e}^{*}θe∗​是IF的值

当达到切换时间时，IF输出的θ∗\theta^{*}θ∗到观测器输出的θ^\hat{\theta}θ^在切换的时候，我们使用惯性环节去做：1Tc∗s+1\frac{1}{Tc*s+1}Tc∗s+11​

假设这里我们的关系环节是 10.01∗s+1(θ^−θe∗)\frac{1}{0.01*s+1} (\hat{\theta}-\theta_{e}^{*})0.01∗s+11​(θ^−θe∗​)，我们可以使用matlab进行离散化

sys=tf(1,[0.01 1])
zh = c2d(sys,0.0001,'zoh') %我们系统是10k
[num,den]=tfdata(zh,'v')

如果出现tf未定义，安装control system 工具箱即可 解释一下下面这个代码：

theta_comp = 0.009950166250832*last_error + 0.990049833749168*last_comp;

首先根据公式

θcomp=1Tc∗s+1(θ^−θe∗)\theta_{comp} = \frac{1}{Tc*s+1} (\hat{\theta}-\theta_{e}^{*})θcomp​=Tc∗s+11​(θ^−θe∗​)
